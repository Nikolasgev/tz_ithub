# Currency App

Данное приложение позволяет просматривать курсы валют, используя API Центрального Банка Российской Федерации.


## Основные функции

### 1. Отображение списка всех валют

На главном экране отображается список всех валют. Вы можете видеть краткую информацию о каждой валюте, такую как код, название и текущий обменный курс. Валюты отсортированы по алфавиту.

### 2. Подробная информация о валюте

При нажатии на определенную валюту вы переходите на экран с подробной информацией о ней. Здесь вы можете увидеть более подробную информацию о валюте, такую как номинал и предыдущий обменный курс.

### 3. Конвертация валют

Была добавлена функция конвертации валют. Теперь вы можете ввести сумму в рублях и узнать, сколько выбранной валюты вы получите за эту сумму.

### 4. Добавление валют в избранное

Теперь у вас есть возможность добавлять валюты в избранное. Когда валюта добавляется в избранное, она появляется на отдельном экране, где вы можете видеть только ваши предпочтительные валюты.

### 5. Светлая и темная тема

Приложение поддерживает светлую и темную темы. Вы можете переключать тему в настройках приложения.

### 6. Оффлайн режим

Приложение сохраняет последние загруженные данные и работает в оффлайн-режиме, если нет подключения к интернету. Это позволяет вам видеть последние доступные данные о валютах.

## Использованные пакеты

- `http`: Для выполнения сетевых запросов к API ЦБ РФ.
- `shared_preferences`: Для кеширования данных о валютах.
- `package_info_plus`: Для отображения информации о текущей версии приложения.

## Как использовать

1. Убедитесь, что у вас установлены все зависимости. Если нет, запустите `flutter pub get`.
2. Запустите приложение с помощью `flutter run`.
